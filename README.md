

Run:
1. python -m venv venv
2. venv\Scripts\activate (Windows) or source venv/bin/activate (Unix)
3. pip install -r requirements.txt
4. python -m app.main
5. Visit http://127.0.0.1:5000

Notes:
- Auto-kill is OFF by default.
- Collect logs (logs/system_data_behavioral.csv) then run train_models_v8.py to train models.
- This build includes tabs: Processes, Analytics, Security, System Specs, About Us
